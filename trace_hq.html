API Key Setup
Protect your Anthropic API key
⚠ Never paste your API key into the app HTML file. Anyone who views source can steal it and charge calls to your account. The correct setup routes all API calls through your own server where the key lives as an environment variable — invisible to the browser.
Step 1 — Store Key in Railway
  1. Deploy trace_backend_package.html → download server.js + package.json
  2. Push both files to a private GitHub repo
  3. Railway.app → New Project → Deploy from GitHub
  4. Railway Dashboard → Variables → Add ANTHROPIC_API_KEY
  5. Add ALLOWED_ORIGIN = your domain
  6. Copy your Railway URL below
Step 2 — Your Railway URL
Railway proxy URL
This is used for status checks. Your key never leaves Railway's servers.
Step 3 — Update App HTML
✓ Once your proxy is running, change the fetch URL in trace_v30.html from https://api.anthropic.com/v1/messages to https://YOUR-RAILWAY-URL/analyse and remove the x-api-key and anthropic-version headers. The proxy adds them server-side.

Sacred geometry reveals the mathematical harmony underlying great artworks. Upload an image and toggle overlays to see how masters used these proportions.
About Sacred Geometry in Art

The golden ratio (φ ≈ 1.618) appears throughout Renaissance and Classical art — from Leonardo da Vinci's Vitruvian Man to Botticelli's Birth of Venus. The ratio creates compositions that feel naturally harmonious to the human eye.

Dynamic symmetry (root rectangles) was used by ancient Greek architects and artists like Georges Seurat. Radial harmony organizes composition around a central focal point, common in religious art and mandalas.

Use overlays above to discover hidden structures in your artwork collection. TRACE can detect these mathematically in analysis results.

🔬 Forensic Analysis Pipeline
Multi-stage scientific analysis suite for Old Master artworks

The forensic analysis pipeline runs 7 stages of examination on the current artwork:

  1. Macro Visual Analysis — Composition, brushwork, light treatment
  2. Pigment Spectroscopy — Period-appropriate pigment identification
  3. Dendrochronology — Panel/canvas dating estimation
  4. Craquelure Pattern Analysis — Age and drying crack patterns
  5. Infrared Reflectography — Underdrawing and pentimenti detection
  6. XRF Spectral Correlation — Elemental composition analysis
  7. UV Fluorescence — Varnish and restoration mapping

🖥 Digital Fingerprinting
Artwork fingerprinting, similarity matching, and stolen database verification

Digital fingerprinting uses perceptual hashing to create a unique identifier for artwork images. This enables:

  • Similarity Matching — Find visually similar works in the database
  • Stolen Art Verification — Cross-reference against stolen art databases
  • Provenance Chain Hashing — Cryptographic hash of the ownership chain
