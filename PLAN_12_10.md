## Known Gotchas

1. **E2E tests**: Playwright manages its own server. The subscribe endpoint bypasses admin token check on localhost (checked via Host header). If a `.env` file is present with `ADMIN_SECRET` set, it's ignored by the test runner.
2. **Provenance tests**: SPARQL queries hit real Getty endpoints — timeout is 20s. Deterministic mock data is used for INTERPOL/ALR/AAMD (no `Math.random()`).
3. **AI provider**: Server auto-detects provider by checking which API key is set. Gemini free tier supports ~60 req/min and includes vision.
4. **Subscription auth**: Localhost bypass requires either no Host header or Host containing 'localhost' or '127.0.0.1'.
